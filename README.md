## О ПРОЕКТЕ

Сайт сделан для друга и для личных полевых испытаний в коде. За оформление так же отвечал друг. Обычный сайт с карточками товара, поисковым окном, рабочими категориями и системой покупок. Для добавления новых товаров используется админ-панель.

## КАК ПОЛЬЗОВАТЬСЯ ПРОЕКТОМ?

Проект никуда не заливался, использовал локальный хостинг чреез xampp. Чтобы код заработал, сперва надо либо самостоятельно создать БД либо использовать заготовленную мою, но без админки.
